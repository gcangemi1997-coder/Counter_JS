# Counter JS

A simple interactive counter web app built with vanilla JavaScript, HTML, and SCSS.

## 🚀 Live Demo

[counterjs3.netlify.app](https://counterjs3.netlify.app/)

## 📋 Features

- Increment, decrement and reset the counter
- Save the current value and display it separately with doubleclick function
- Fully responsive design (mobile, tablet, desktop)
- DOM built dynamically with JavaScript
